== Description ==

Newseqo is a Clean, Easy to Use News Magazine WordPress Theme for Newsportal, Magazine site, Publishers, Blogs, Online, Editors, and Business News. This amazing News Magazine theme built with Bootstrap4 framework, Elementor Builder for a perfect News Magazine Website.

Newseqo comes with 3 Header Styels, 4 Home Variations and 6+ News Blocks such as News Tab, News Slider, News List, News Carousel etc. Here is Live preview: https://demo.themewinter.com/wpsite/newseqo/ and Documentation: https://support.themewinter.com/docs/themes/docs-category/newseqo/


== Options ==

All options are in the Customizer.
2 Header Styles 

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.
